LD4 2022 Conference Workshop

Visualizing and Training Machine Learning Models with Sinopia Linked Data

Introduction

This work-shop will introduce you to downloading and exploring the RDF created in the Sinopia Linked Data Editing environment. We will then build upon these Sinopia data artifacts we created by applying various machine learning technologies and techniques for such tasks as FAST subject heading and template classification. Finally, we will discuss Data Statements and Model Cards and how to apply them to the work we did today.

Workshop Schedule

This workshop will be broken down into four parts, with breaks during the sessions.

1. Introduction, Setup, Analysis, and Visualization of Sinopia RDF

  • RDF Graph Explorer
    1. Loading RDF into Graphs
    2. Analysis of RDF with Pandas
    3. Visualization of RDF

2. Using spaCy and HuggingFace Natural Language Processing (NLP)

3. PyTorch with FastAI

4. Increasing transparency with Model Cards and Data Statements
